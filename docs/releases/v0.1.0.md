# Hydra Display v0.1.0

**The first public release of Hydra Display** — a free, open-source, native macOS app
for creating virtual displays, with a Liquid Glass interface built for macOS 26 (Tahoe).

## ✨ Highlights

- **Create virtual displays in seconds.** Pick a name, toggle HiDPI/Retina, choose the
  resolutions to advertise, and it appears to macOS as a real monitor.
- **Menu bar quick actions.** Spin up a 4K Retina, 1440p, or 1080p display with one
  click — no need to open the main window.
- **Mirror & arrange.** Drag displays to rearrange your desktop space and mirror any
  screen onto another. Works for physical *and* virtual displays.
- **Native Liquid Glass design.** Content on the material layer, glass reserved for the
  floating controls — following Apple's macOS Tahoe guidance.
- **No Developer ID required.** Builds and runs with ad-hoc "Sign to Run Locally".

## 📦 What's included

- Create / remove virtual (HiDPI) displays via the private `CGVirtualDisplay` API,
  wrapped defensively with runtime availability checks.
- Resolution presets: 16:9, 16:10, ultrawide, portrait — plus custom resolutions.
- Mirroring and desktop arrangement via the public `CGConfigureDisplay*` API.
- A custom About window and a single `AppInfo` source of truth for the app's edition.
- Apple-style app icon and `systemOrange` accent.

## ✅ Requirements

- **macOS 26.0 (Tahoe)** or later.

## ⬇️ Install

1. Download **`HydraDisplay.app.zip`** below and unzip it.
2. Move **`HydraDisplay.app`** to `/Applications`.
3. The app is **not notarized** (it uses a private API and ships without a Developer
   ID), so Gatekeeper will warn on first launch. Either:
   - **Right-click → Open**, then confirm, **or**
   - run:
     ```bash
     xattr -dr com.apple.quarantine /Applications/HydraDisplay.app
     ```

## ⚠️ Known limitations

- Virtual displays are **not persisted** across app launches yet.
- Creating displays relies on a **private Apple API** — it cannot ship on the Mac App
  Store and may break on a future macOS release. See
  [docs/PRIVATE_API.md](../PRIVATE_API.md).
- The UI is **English-only** for now.
